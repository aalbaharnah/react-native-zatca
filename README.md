# react-native-zatca

A react native library to generate ZATCA compliant e-invoices for Phase II. 

## Installation

```sh
npm install react-native-zatca
```

## Usage


```js
import { multiply } from 'react-native-zatca';

// ...

const result = await multiply(3, 7);
```
